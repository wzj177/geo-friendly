What aeo.js Generates

Discover the 8 file types that aeo.js generates to make your site AI-discoverable.

aeo.js generates 8 types of files from your existing content, each targeting a different aspect of AI discoverability:

  • robots.txt — AI-aware directives that allow GPTBot, ClaudeBot, Bingbot, and other AI crawlers to index your content. Includes sitemap references and crawl delay settings.
  • llms.txt — A concise summary of your site optimized for LLM context windows. Contains site metadata, page list with descriptions, and navigation structure.
  • llms-full.txt — Complete content export with full markdown text for every page. Designed for AI systems that need deep content understanding.
  • sitemap.xml — Standard XML sitemap with all discoverable pages, last-modified dates, and change frequency hints for search engines and AI crawlers.
  • ai-index.json — Structured JSON index with page content, descriptions, keywords, and metadata — purpose-built for AI ingestion pipelines.
  • schema.json — Schema.org JSON-LD structured data including WebSite, Organization, Article/WebPage, BreadcrumbList, and auto-detected FAQPage schemas.
  • docs.json — Documentation manifest with categorized content, markdown URLs, and content summaries for AI documentation assistants.
  • Per-page .md files — Individual markdown files for each page with YAML frontmatter, clean content extraction, and structural preservation.
