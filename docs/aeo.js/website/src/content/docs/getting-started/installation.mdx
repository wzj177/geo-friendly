## Install the package

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install aeo.js
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add aeo.js
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add aeo.js
    ```
  </TabItem>
</Tabs>

## Or use the CLI directly

No installation needed — run it with `npx`:

```bash
npx aeo.js generate
```

## Requirements

- **Node.js** 18 or later
- A framework with a build step that outputs HTML (or use the CLI for any static site)
