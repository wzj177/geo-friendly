# 快速开始

本指南将帮助你快速开始使用 VitePress 和 Geo-Friendly。

## 安装

```bash
npm install
```

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 生成 GEO 文件

```bash
npm run geo:generate
```

## 配置

编辑 `geofriendly.yaml` 来自定义生成选项。
